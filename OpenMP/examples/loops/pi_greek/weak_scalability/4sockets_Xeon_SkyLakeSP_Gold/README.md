### Index of files

In the following $T_1$ = run-time with 1 thread, $T_N$ = ru-time with N threads

`scalability_normalized_all.png` → all timings, normalized as $T_N / T_1$ with +- stddev
`efficiency.VV.png` → timings rescaled as $T_1 / T_N$ for verion $VV
`scalability_normalized.VV.png` → timings normalized as $T_N / T_1$ for verion $VV
`scalability.VV.png` → timings for verion $VV
